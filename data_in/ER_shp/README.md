# Source 


- [ISTAT - Confini delle unità amministrative a fini statistici](https://www.istat.it/notizia/confini-delle-unita-amministrative-a-fini-statistici-al-1-gennaio-2018-2/)
- Data accessed on November 8, 2025
- Data updated to January 1, 2025

- Content:the shapefiles of the administrative boundaries of Italy updated as of **January 1, 2025**.

+ Il formato dei dati è shapefile nel sistema di riferimento WGS84 ( vedi file file apposito (formato prj))

+ Data have been filtered into subsets relative to Parma only, and saved in the R data folder as sf objects for easier use in R.
